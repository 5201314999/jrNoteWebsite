# md 文档编写 😄

# 前言

Markdown 的理念是，能让文档更容易读、写和随意改。HTML 是一种发布的格式，Markdown 是一种书写的格式。

1 现在有很多md 编辑工具，直接界面操作就可以编写文档，本地记录的语法是采用文本编写时需要用到的。 2 不同编辑器语法也是有所偏差的 3 本文的语法比较核心和通用

# 语法

不同工具快捷键不一样，但功能类似，写法一样的,
